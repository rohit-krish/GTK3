## GTK in C

### Install

```sh
sudo apt install libgtk-3-dev
```

### Run Program

```sh
gcc `pkg-config --cflags --libs gtk+-3.0` -o your_file your_file.c
```

### Docs

```
https://devdocs.io/gtk~3.20/
```

## GTK in C++

### Install

```sh
sudo apt install libgtkmm-3.0-dev
```

### Run Program

